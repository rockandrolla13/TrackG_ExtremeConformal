This step investigated free historical data sources for FX and cryptocurrency markets. For FX data, HistData.com was identified as the preferred source, offering 1-minute data for the required pairs and date ranges. A Python script (`src/data/download_fx.py`) was created to automate the download process by simulating form submissions, though it may require adjustments if the website's structure changes.

For cryptocurrency data, CryptoDataDownload.com was selected as the most promising source. However, the availability of free, no-registration 1-minute data is ambiguous. A download script (`src/data/download_crypto.py`) was created based on a guessed URL structure, which serves as a starting point but may need manual verification and correction. Both scripts, along with documentation and lists of key event dates, have been added to the repository to facilitate the data acquisition phase.
